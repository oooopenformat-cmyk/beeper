Програмка для Андроид 8 и выше
Общение между пользователями при помозщи азбуки морзе
Общение идет черз php файл ( его можно запустить на своем сервере) и прописать его в настройках программы
Все те кто уставовит программу на смартфон и подключиться к php файлу тот и будет "слышать" и сможет отправлять сообщения для всех кто подключился к томуже php файлу
Php файл который можно установить на сервер возможно размещу в  это м репозитории
